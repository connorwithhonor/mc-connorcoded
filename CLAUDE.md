# Mission Control Dashboard

## Purpose
Connor's daily mission control. Opens on screen 1 (left 32"). Shows all revenue lanes with ship status, next moves, money math. Glanceable from a head turn.

## Voice / Style
- Dark Tron aesthetic (cyan, gold, red on deep dark)
- No em dashes anywhere
- Short lines. Boss energy.
- Next move = THE ONE thing that moves the lane forward
- No cluttered nice-to-haves

## Update Protocol
After every significant milestone or pivot, update the lane data in index.html. Push to mempalace as well so the state is versioned.
